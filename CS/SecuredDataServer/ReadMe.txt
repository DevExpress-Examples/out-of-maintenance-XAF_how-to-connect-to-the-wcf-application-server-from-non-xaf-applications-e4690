Project Description

The Server project implements an XAF Middle Tier application 
server. Modify the app.config file to setup data connection for server and client. 
Edit the ApplicationServerService.cs(vb) file to add client application’s modules
and set up the server-side security. Design the ProjectInstaller.cs(vb) file to
configure service installation.
